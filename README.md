# Prompt Engineering Research

## Objective

This project looks at a subset of the 26 prompt engineering principles from Bsharat et al. and tests whether they still make a difference on modern LLMs. The original principles were designed to get better responses out of language models, but models have changed a lot since then. The goal is to see which of these principles still hold up and which ones don't really matter anymore.

## Background

Bsharat et al. published a set of 26 principles for writing better prompts for LLMs. This project takes around 6 of those principles and tests them against current models to see if they still improve output quality compared to prompts that don't use them.
